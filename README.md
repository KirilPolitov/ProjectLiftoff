Project Liftoff

2d space simulation game built using the C++ libraries of SFML

Features:
-Newtonian gravitation and orbital mechanics and N-Body simulation
-Exponential atmospheric pressure using the simplified barometric formula
-Celestial body collisions
-Sound FX and ambient music
-Keyboard controlled rocket

Keybindings:
-W-accelerate forward
-S-accelerate backward
-A/D-rotate counterclockwise/clockwise
-Z/X-zoom in/out

Requirements:
-All files provided in the release to be in the same folder
-Windows PC or Linux with WINE
-X86-64 CPU or an emulator for it
